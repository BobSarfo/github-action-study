## GitHub Pipeline Actions
[![Build Status](https://github.com/BobSarfo/github-action-study/actions/workflows/pr-workflow.yml/badge.svg)](https://github.com/BobSarfo/github-action-study/actions/workflows/pr-workflow.yml)


This repository contains a simple .NET 8.0 project that demonstrates the use of GitHub Actions for continuous integration and deployment (CI/CD). The project includes a workflow that builds and tests the application whenever changes are pushed to the repository.
